# Labels

This repository uses behavior-oriented labels.

Labels reflect:
- agent behavior impact
- prompt lifecycle changes
- evidence quality
- recycle outcomes

## Core Labels
- abd-core: paradigm level change

## Behavior Labels
- agent-behavior: behavior-level issue
- prompt-change: prompt update
- prompt-patch: corrective behavior patch

## Quality Labels
- evidence-required: missing or weak evidence

## Recycle Labels
- recycle-output: kept/reused/banned required
- anti-pattern: banned behavior identified
- pattern-promotion: reusable behavior promoted

## Risk Labels
- conflict-detected: conflict identified
- high-risk: needs careful review
